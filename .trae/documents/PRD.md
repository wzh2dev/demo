## 1. 产品概述

基于高德地图的快餐店等级评价展示页面，用户通过手动编辑 JSON 文件录入快餐店数据，在地图上以不同颜色标记展示快餐店的等级与评价信息。

- 目标用户：需要快速了解快餐店质量等级的个人用户
- 核心价值：以直观的地图可视化方式呈现快餐店评价，一目了然

## 2. 核心功能

### 2.1 功能模块

1. **地图主页**：高德地图展示快餐店点位，点击弹出信息窗口

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 地图主页 | 地图容器 | 高德地图浅色模式，中心点南京，不显示景点 |
| 地图主页 | 标记点 | 根据快餐店等级用5种颜色展示标记点 |
| 地图主页 | 信息弹窗 | 点击标记点弹出该快餐店的等级与评价信息 |
| 地图主页 | 图例面板 | 展示5个等级与对应颜色的图例说明 |

## 3. 核心流程

用户打开页面 → 地图加载并以南京为中心 → 读取 JSON 数据 → 在地图上渲染彩色标记点 → 用户点击标记点 → 弹出信息窗口展示等级与评价

```mermaid

    "渲染彩色标记点" --> "用户点击标记点"

```

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深灰底色 + 等级色彩系统（绿/蓝/橙/黄/红）
- 按钮风格：圆角胶囊按钮，微阴影
- 字体：思源黑体 / Noto Sans SC，标题 18px 加粗，正文 14px 常规
- 布局风格：全屏地图 + 左上角图例浮层 + 点击弹窗
- 图标风格：简洁圆形标记点，颜色区分等级

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 地图主页 | 地图容器 | 全屏高德地图，浅色模式，无景点标注 |
| 地图主页 | 标记点 | 圆形彩色标记，5种颜色对应5个等级 |
| 地图主页 | 信息弹窗 | 白色卡片弹窗，显示等级标签（彩色）+ 评价文字 |
| 地图主页 | 图例面板 | 左上角半透明浮层，5行等级+颜色对应 |

### 4.3 响应式

- 桌面优先，地图全屏自适应
- 移动端适配：图例面板缩小，弹窗宽度自适应

## 5. 数据结构

### 5.1 快餐店等级定义

| 等级 | 颜色 | 色值 |
|------|------|------|
| 夯 | 绿色 | #22C55E |
| 顶级 | 蓝色 | #3B82F6 |
| 人上人 | 橙色 | #F97316 |
| npc | 黄色 | #EAB308 |
| 拉完了 | 红色 | #EF4444 |

### 5.2 JSON 数据格式

```json
[
  {
    "id": 1,
    "lng": 118.79,
    "lat": 32.06,
    "level": "夯",
    "rating": "味道很正，分量足，性价比高"
  }
]
```

> 注意：不需要店铺名称字段
